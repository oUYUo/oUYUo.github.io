简介

什么是计算符号（Symbolic Computation)

符号计算就是以象征性的方法处理数学对象的计算

意思就是python包里面的math方法，开更号的话就是一个有限小数，不准确，只是大概。但是sympy的话结果是 2*sqrt（2） 是一种指代，象征就这个意思。他可以准确 代表8的开更号。

表达式保持不变

他没有变为x^2+2xy，而是保持不变。

计算威力

符号计算系统 (如 SymPy) 的真正威力是象征性地进行各种计算的能力。SymPy 可以简化表达式, 计算导数, 积分和极限, 求解方程, 使用矩阵, 多, 更多, 并以象征性的方式进行。它包括用于绘图、打印的模块 (如2D 漂亮的数学公式打印输出, 或代码生成, 物理, 统计, 组合, 数字理论, 几何, 逻辑等。

变量定义

symbols定义好之后，之后对变量的改变不会改变表达式

若要改变表达式中更改符号的值，使用subs

等于符号

python语言里==表示判断两个值是否相等。sympy没有改变它。所以sympy写一个等式用Eq

这种比较，只有在结构上完全相等时，两个表达式才会比较。
如果比较（x+1)^2 和x^2+2x+1就不行
我觉得这个主要用在设置方程式

simplify

最后两个注释：^ 和 /

当组合python和sympy对象时，会得到一个sympy对象

基本操作

替换

将表达式里所有实例替换成其他内容

替换有两个原因：

  • 一个是想得到他的值，把未知数替换
  • 将子表达式替换为另一个子表达式
  • 第二个是, 如果我们想要执行一个非常控制的简化

他返回一个新的表达式，sympy对象时不可变的。

若要一次执行多个替换，将列表穿个他。subs(old,new )

通过判断语句把指数为双数的未知数换成y

将字符串转为sympy表达式

  • 用simplfy
  • 将数值表达式计算为浮点数字，用evalf

有时, 在计算表达式后, 舍入错误会小于所需的精度。通过将标志设置为 True, 可以在用户的酌情权下删除此类数字。chop
